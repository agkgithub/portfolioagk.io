MOVIE COLLECTION REPORT DASHBOARD

In this project, I have created a dashboard of the movies KGF Chapter 1 and Chapter 2 collection report. For this, I have gathered the data of first 6 weeks daywise and regionwise revenue, which is collected from several online sources and displayed under different categories like Weekly collection, Collection by region and its percentage, profit earned etc.

SALES DATA DASHBOARD

In this project, I have made a sales data dashboard using an open-source dataset of an online store that has Furniture, Office supplies and Technological products which are sold under different sub-categories, ordered under different priority conditions in United States from Jan to June 2015.

EXPLORATORY DATA ANALYSIS USING PYTHON

In this project, I took a Four wheeler vehicle sales company dataset which is open source and explored the data based on various categories and elements associated according to customer's perspective and made graphical representations of different variables using Python programming.
